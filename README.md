# R-Air-Pollution-Analysis

## "Developed a suite of R functions to process and analyze atmospheric particulate matter data from 332 monitoring stations. The project includes automated data acquisition and cleaning, with a primary focus on calculating the correlation between sulfate and nitrate levels based on a user-defined threshold for complete observations."
